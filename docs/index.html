Before you begin

This is a Xamarin.Forms class. To complete the coding exercises, you will need a Xamarin.Forms development environment with either an emulator or a device to run the exercise on. Please see the setup page if you need help installing the Xamarin development environment.

Overview

In this class you will learn about some of the newer thread synchronization capabilities that have been added into the platform. This is a continuation of CSC352 - An introduction to Thread Safety and Synchronization where we introduced the topic. If you have not taken that class, please do so before attending this one.

Objectives

  1. Beyond Monitor - implement other synchronization strategies
  2. Work with Synchronized Collections

Lecture

CSC353 - More about Synchronization.

Demos

  1. Use a semaphore to control access to shared objects
    In this demo, we will use a Semaphore to limit the number of threads that can access a resource. The completed project is avaliable in Demos folder.
  2. Use Blocking Collections to provide an async search experience
    In this exercise, we will use a blocking collection to provide an asynchronous search operation in our application. he completed project is avaliable in Demos folder.
